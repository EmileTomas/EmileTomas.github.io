Visual Studio 2015下MPI环境配置

Posted on 2016-10-13   |   In Tutorials   |     |   visitors

  1. 在微软的MPI下载地址中MS-MPI downloads目录下下载MS-MPI内两个安装包并安装。
    分别是：

    • msmpisdk.msi
    • MSMpiSetup.exe

  2. 在VS中创建一个C++工程，选择为空项目。

  3. 进入项目-> 项目属性中，点击C/C++ ，在附加包含目录下添加

    • C:\Program Files (x86)\Microsoft SDKs\MPI\Include;

    • C:\Program Files (x86)\Microsoft SDKs\MPI\Include\x64;


      注：如果是32位程序则添加相应的X86目录即可

  4. 在链接器-> 所有选项中附加库目录中添加

    • C:\Program Files (x86)\Microsoft SDKs\MPI\Lib\x64;

      在附加依赖项末尾添加

    • msmpi.lib;


  5. 添加示例程序
    ​

  6. 启动程序，会发现只打印一条数据，并没有模拟并行化。

模拟群集运行

  1. 打开命令行界面，进入项目目录下的/x64/Debug子目录
  2. 执行mpiexec.exe -n 4 项目.exe
  3. 显示结果

Open Command Line 插件

如何在Visual Studio 下快速打开项目路径，省去cd命令，这里推荐Open Command Line 插件
下载地址

配置

  1. 安装完成后，重启Visual Studio，在右侧项目窗口右键，选择Open Command Line，点击Setting
  2. 在选项窗口中的Command arguments 写入/k cd ../x64/Debug，点击确认
  3. 在Visual Studio 主界面使用快捷键Alt+Space 即可快速进入程序相应目录
  4. 输入mpiexec.exe -n 4 项目.exe 。
    显示结果
