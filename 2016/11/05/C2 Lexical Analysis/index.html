C2 Lexical Analysis

本教程对编译原理词法分析部分实验进行说明，旨在解决一些配置上的问题，以帮助他人少走弯路。

虎书给出的所有framework都是默认以makefile进行编译的,即在makefile目录下终端输入make命令进行编译.但是由于助教给出的词法工具是Jflex-1.4.1版本,而makefile中指定的是Jlex,这就导致make时出现错误.

解决办法:

  1. 前往官方下载最新版JFlex-1.6.1.tar.gz, 下载当前最新版本即可,此刻最新是1.6.1

  2. 解压至任意目录下,推荐解压至不会轻易删除的目录中,比如我解压到了/usr/share目录下.

  3. 添加Jflex-1.6.1下的/lib/jflex-1.6.1.jar进Linux的CLASSPATH目录中.

    实例方法:

    1. sudo gedit /etc/profile
    2. 在末尾export CLASSPATH=一行中加入:/usr/share/jflex-1.6.1/lib/jflex-1.6.1.jar(注意开始的:号.是分隔用的)

  4. Log out 以让CLASSPATH生效

  5. 修改Makefile中的Jlex部分,这里贴出修改后的内容,请自行寻找区别
